Un "Cheat Sheet" Para VisiData
Versión VisiData v3.0.2
Actualizado 2024-03-08
Más info visidata.org
jsvine.github.io/intro-to-visidata
Obtener ayuda
Ctrl-h Abrir el menú de ayuda
z + Ctrl-h Mostrar lista de comandos (para la hoja actual)
Resolver problemas
U/R Deshacer/Rehacer
Ctrl-c Abortar comando actual
q Salir de hoja ("sheet") actual
Ctrl-q Forzar salida de VisiData
Entrada / salida
o + archivo Abrir un archivo
Ctrl-s + archivo Guardar hoja actual a nombre del archivo
gY/gzY Copiar (columna de) filas seleccionadas al portapapeles
Metahojas
S Hoja de hojas ("Sheets Sheet")
gS Cementerio de hojas
C Hoja de columnas
O Hoja de opciones globales
zO Opciones de hoja actual
Mover el cursor ...
gj a la última fila
gk a la primera fila
gh a la última columna de la izquierda
gl a la última columna de la derecha
Ctrl-f una página hacia abajo (adelante)
Ctrl-b una página hacia arriba (atrás)
Moverse usando el buscador
/ + regex Buscar hacia adelante en la col. actual
? + regex Buscar hacia atrás en la col. actual
g/ + regex Buscar hacia adelante en todas las columnas
g? + regex Buscar hacia atrás en todas las columnas
n Ir a la siguiente fila que coincida
N Ir a la anterior fila que coincida
Ordenar las filas ...
[ ascendentemente, de acuerdo con columna actual
] descendentemente, de acuerdo con columna actual
g[ ascendentemente, de acuerdo con columnas claves
g] descendentemente, de acuerdo con columnas claves
Selección básica de filas
s Seleccionar la fila actual
u Deseleccionar la fila actual
t Cambiar la fila actual entre seleccionada / deseleccionada
gs Seleccionar todas las filas
gu Deseleccionar todas las filas
gt Invertir selección / deselección de todas las filas
Selección avanzada de filas
| + regex Seleccionar las filas donde regex coincide con la columna actual
\ + regex Deseleccionar las filas donde regex coincide con la columna actual
g| + regex Seleccionar las filas donde regex coincide con cualquier columna
g\ + regex Deseleccionar las filas donde regex coincide con cualquier columna
, Seleccionar las filas donde la columna actual coincide con la celda actual
g, Seleccionar las filas donde cualquier columna coincide con la celda actual
z| + expr Seleccionar las filas donde expr se evalúa como True
z\ + expr Deseleccionar las filas donde expr se evalúa como True
Mover filas y columnas
J Mover fila hacia abajo
K Mover fila hacia arriba
H Mover columna hacia la izquierda
L Mover columna hacia la derecha
Establecer tipos de columna
# Entero
% Flotante
$ Moneda
@ Fecha
~ Texto
Renombrar columnas
^ Renombrar columna actual
g^ Establecer como nombres de todas las columnas sin nombre los valores de la fila actual (o selección)
gz^ Establecer como nombres de todas las columnas los valores en la fila actual (o selección)
Cambiar tamaño de las columnas
_ Ajustar columna actual para encajar el texto de las filas visibles
g_ Ajustar todas las columnas para encajar el texto de las filas visibles
z_ + n Cambiar anchura de la columna actual a n caracteres
z- Reducir anchura de la columna actual a la mitad
- Ocultar columna actual cambiando su anchura a 0
gv Revelar todas las columnas
Filtrar
" Crear una hoja nueva con las filas seleccionadas
gz" Crear nueva hoja con copia "profunda" de las filas seleccionadas
Resumir los datos
F Crear tabla de frecuencias de la columna actual
gF Crear tabla de frecuencias de las columnas claves
+ + agr Añadir un agregador a la columna
z+ + agr Calcular (una vez) el promedio, la suma, etc.
I Crear "hoja de descripción" con estadísticas básica de cada columna
Crear nuevas columnas
za Crear columna vacía
i Crear columna que incrementa (1,2,3…)
= + expr Crear columna nueva con resultado de expr en Python evaluada por cada fila
: + regex Crear columna dividiendo la columna actual de acuerdo con regex
; + regex Crear columna extrayendo los grupos regex de la columna actual
( Expandir columna
' Crear una copia "congelada" de la columna actual
Cambiar la forma de los datos
W Crear una tabla dinámica, con columnas claves como filas y columna actual como valores
M Crear una hoja "fundida" o "larga", respetando las columnas claves
T Crear un hoja transpuesta, intercambiando las columnas y filas
Editar las celdas
e Empezar a editar la celda actual
Enter Terminar la edición
Ctrl-c Cancelar la edición
Ctrl-a Ir al pricipio de la celda
Ctrl-e Ir al final de la celda
Ctrl-k Borrar el contenido que sigue al cursor
Copiar y pegar internamente
y/gy Copiar la fila actual (o selección)
p Pegar filas previamente copiadas
zy/zp Copiar/pegar la celda actual
gzy/gzp Copiar/pegar la columna de filas seleccionadas
zP/gzP Pegar desde portapapeles a la celda actual (o selección)
Misceláneo
! Marcar la columna actual como "clave"
Ctrl-r Recargar la hoja
Ctrl-^ Saltar entre hoja actual y anterior
Space Abrir entrada para comandos de "nombres largos"
